# 0.6: New note in single page app diagram

```mermaid
sequenceDiagram
    participant browser
    participant server

    Note right of browser: User writes a note and clicks "Save"
    Note right of browser: spa.js prevents default form submit

    Note right of browser: SPA updates the UI immediately
    Note right of browser: (adds the new note to the list)

    browser->>server: POST https://studies.cs.helsinki.fi/exampleapp/new_note_spa
    activate server
    Note right of server: Server stores the new note
    server-->>browser: 201 Created (JSON of the created note)
    deactivate server

    Note right of browser: SPA confirms success
    Note right of browser: (no page reload, notes stay visible)
```
